## Run Locally

Clone the project

```bash
  git clone [https://link-to-project](https://github.com/ViquinhoDev/TABOI-craft)
```

Go to the project directory

```bash
  cd TABOI-craft
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
